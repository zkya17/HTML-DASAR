Apa itu Jurusan Pengembangan Perangkat Lunak dan Gim (PPLG)?

PPLG adalah singkatan dari Pengembangan Perangkat Lunak dan Gim dan merupakan sebuah jurusan yang ada di Sekolah Menengah Kejuruan (SMK). PPLG adalah sebuah jurursan yang mempelajari dan mendalami semua cara-cara pengembangan perangkat lunak termasuk pembuatan, pemeliharaan, manjemen organisasi pengembangan perangkat lunak dan manajemen kualitas.

Apa Saja yang Dipelajari di Jurusan PPLG?

Sebetulnya PPLG (Pengembangan Perangkat Lunak dan Gim) itu Gampang! tidak jauh jauh sama 3 hal ini yaitu:

  1. Coding
  2. Desain
  3. Alogaritma

Didalam Coding, Desain dan Alogaritma tersebut masih ada point-point penting yang dipelajari di jurusan PPLG (Pengembangan Perangkat Lunak dan Gim)

Jurusan PPLG Kerja Apa?

Banyak yang tanya jurusan PPLG itu nanti kalau lulus bisa kerja apa? Apa saja lowongan kerja lulusan AMK jurusan PPLG. Berikut daftar peluang kerja jurusan PPLG:

    • Developer IT
    • Progammer
    • Web Engineer
    • Game Developer
    • Database Engineer / data base administrator
